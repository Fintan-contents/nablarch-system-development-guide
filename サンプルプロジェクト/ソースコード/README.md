# サンプルソースコード

サンプルソースコードは、以下の2つのプロジェクトにより構成されます。

| プロジェクト   | 役割                                                           |
|----------------|----------------------------------------------------------------|
| proman-project | プロジェクト管理システム （**Pro**ject **Man**agement System） |
| climan-project | 顧客管理システム （**Cli**ent **Man**agement System）        |
