# ビルド方法

## 動作環境

動作環境は[開発環境構築ガイド](../../../サンプルプロジェクト開発ガイド/PGUT工程/開発環境構築ガイド.md)を参照してください。

PostgreSQLの初期ユーザーとパスワードは以下の通りに設定します。

| ユーザー| パスワード |
|:------|:--------|
| postgres | password|

## ビルド方法

以下のコマンドを実行してください。
```
$mvn -P gsp clean generate-resources
```

注意：アプリケーションを起動する前に、必ず上記のコマンドを実行しておいてください。
