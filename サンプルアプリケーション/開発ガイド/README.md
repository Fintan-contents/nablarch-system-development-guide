   [collaborage→プロジェクトの開発準備→ConcourseでのCI追加](https://github.com/Fintan-contents/collaborage/blob/master/doc/dev.md#concourseでのci追加)  の「Mavenリポジトリにデプロイするため、distributionManagementの設定を追加します」を参照。
